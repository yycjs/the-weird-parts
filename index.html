Equality or == vs ===

== compares value & truthy- falsyness

1 == 1 // -> true
1 == '1' // -> true
1 == 2 // -> true
'' == false // -> true
[] == '' // -> true
null == undefined // -> true

=== compares value and type

1 === 1 // -> true
1 === '1' // -> false
[] === '' // -> false

ALWAYS USE ===
